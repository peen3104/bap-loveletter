# 📩 Letter Management & Approval System

A web-based letter submission and approval system built using Laravel.  
This application allows staff to create and submit letters which are reviewed and approved/rejected by supervisors in a structured workflow.

## ✨ Features

- User authentication (Login & Register)
- Role-based access (Staff, Supervisor)
- Letter submission system
- Letter approval & rejection workflow
- Status tracking
- Dashboard for each role
- Database persistence using PostgreSQL

---

## 🏗 System Workflow

1. Staff creates a letter.
2. Letter is submitted for review.
3. Supervisor reviews:
   - Approve → Letter marked as approved.
   - Reject → Letter returned to staff for revision.
4. Staff can edit and resubmit rejected letters.
